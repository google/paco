com.google.gwt.typedarrays.shared

Interface Uint16Array

  • All Superinterfaces:
    ArrayBufferView
    All Known Implementing Classes:
    Uint16ArrayImpl, Uint16ArrayNative

    public interface Uint16Array
extends ArrayBufferView
    A view representing an ArrayBuffer as 16-bit unsigned integers. Storing out-of-range values are mapped to valid values by taking the bottom 16 bits of the value. "http://www.khronos.org/registry/typedarray/specs/latest/#7"
