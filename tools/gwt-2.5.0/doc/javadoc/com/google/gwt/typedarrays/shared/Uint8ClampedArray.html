Interface Uint8ClampedArray

    public interface Uint8ClampedArray
extends Uint8Array
    A view representing an ArrayBuffer as 8-bit unsigned integers. Storing out-of-range values are mapped to valid values by clamping to the range 0-255. "http://www.khronos.org/registry/typedarray/specs/latest/#7"
