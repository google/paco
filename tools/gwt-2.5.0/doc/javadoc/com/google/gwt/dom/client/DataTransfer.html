      • setData

        public final void setData(java.lang.String format,
           java.lang.String data)
        Set the data for the specified format to associate with all drag events in the current drag and drop sequence. The data can be read using getData(String) from any subsequent drag events in this sequence (such as the drop event).

        The format is usually the mime-type of the data, but can also be text.

        Parameters:
        format - the format, which is usually the mime-type of the data
        data - the data to associate with the format

        W3C Specification
