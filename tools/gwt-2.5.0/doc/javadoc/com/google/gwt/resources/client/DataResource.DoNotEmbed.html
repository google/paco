com.google.gwt.resources.client

Annotation Type DataResource.DoNotEmbed


  • @Documented
@Retention(value=RUNTIME)
@Target(value=METHOD)
public static @interface DataResource.DoNotEmbed
    Specifies that the resource or resources associated with the ResourcePrototype should not be embedded into the compiled output. This may be useful, for exmaple, when it a particular browser or plugin is unable to handle RFC 2397 data URLs.
