Εύρεση & Συμμετοχή σε Πειράματα

Ο ευκολότερος τρόπος να ξεκινήσετε με το Paco είναι να εγγραφείτε σε ένα υφιστάμενο πείραμα. Υπάρχουν πολλά παραδείγματα.

Από την κυρίως οθόνη, επιλέξτε "Εύρεση Πειραμάτων". Μπορεί να χρειαστεί να επιλέξετε "Ενημέρωση Πειραμάτων" για να λάβετε μια πρόσφατη λίστα πειραμάτων από τον διακομιστή. Όταν δείτε κάτι που σας ενδιαφέρει, πατήστε πάνω του για να δείτε την περιγραφή, τον δημιουργό και το πρόγραμμα δειγματοληψίας.

Εάν νομίζετε ότι θέλετε να λάβετε μέρος σε αυτό το πείραμα, πατήστε "Συμμετοχή στο πείραμα". Αυτό θα σας μεταφέρει στη σελίδα που εξηγεί τον τρόπο διαχείρησης των δεδομένων, από τον δημιουργό του πειράματος.
Σημείωση: Σιγουρευτείτε ότι λαμβάνετε μέρος μόνο σε πειράματα τα οποία διαχειρίζονται τα δεδομένα σας με έναν ικανοποιητικό για εσάς τρόπο.

Εάν αποφασίσετε ότι νιώθετε άνετα με τον τρόπο διαχείρησης των δεδομένων, τότε, τσεκάρετε την επιλογή "Αποδοχή". Ακολούθως, θα ερωτηθείτε για το πρόγραμμα με τις ημέρες/ώρες για το πείραμα. Συνήθως, θα θέλετε να κρατήσετε το αρχικό πρόγραμμα, αλλά πιθανώς να θέλετε να το αλλάξετε ελαφρώς, για να ταιριάζει παραπάνω με την ρουτίνα σας. Πατήστε "Αποθήκευση" για να ολοκληρώσετε την διαδικασία εγγραφής.

Εάν αποφασίσετε ότι δεν θέλετε να λάβετε μέρος, απλά πατήστε το κουμπί επιστροφής, μέχρι να φτάσετε στην σελίδα με τα πειράματα ή στην αρχική σελίδα.

Λαμβάνοντας μέρος σε ένα Πείραμα

Πειράματα σηματοδότησης

Αφού αποθηκεύσετε το πρόγραμμα, συνήθως, το πείραμα θα σας ειδοποιεί περιοδικά για να απαντήσετε στις ερωτήσεις.

Μια ειδοποίηση θα εμφανιστεί στο πάνω μέρος της οθόνης σας, και το τηλέφωνο θα δονηθεί ή θα ηχήσει, αναλόγως με τις ρυθμίσεις που επιλέξατα, για να σας υπενθυμίσει ότι είναι ώρα να απαντήσετε σε ένα πείραμα. Για να απαντήσετε, απλά πατήστε πάνω στην ειδοποίηση για να εμφανιστεί η σελίδα με τις ερωτήσεις. Όταν τελειώσετε, πατήστε "Αποθήκευση απάντησης". Θα εμφανιστεί μία σελίδα επιβεβαίωσης ότι η αποθήκευση ήταν επιτυχής. Για να εγκαταλείψετε, απλά πατήστε το κουμπί επιστροφής.

Πειράματα αυτοαναφοράς

Εάν το πείραμα είναι μόνο αυτο-αναφοράς, δεν θα σας ειδοποιήσει, και μπορείτε να το απαντήσετε οποιαδήποτε στιγμή προτιμείτε. Για να απαντήσετε, πηγαίνετε στο Paco και πατήστε "Τρέχοντα πειράματα" στην κυρίως οθόνη. Αυτό θα σας δείξει τη λίστα με τα πειράματα στα οποία λαμβάνετε μέρος. Πατήστε στο όνομα του πειράματος στο οποίο θέλετε να απαντήσετε. Αυτό θα ανοίξει την φόρμα του πειράματος και θα σας επιτρέψει να απαντήσετε στις ερωτήσεις όπως ακριβώς θα απαντούσατε και στα πειράματα σηματοδότησης, πιο πάνω.

Επεξεργασία Προγράμματος Πειράματος

Εάν θέλετε να αλλάξετε το πρόγραμμα δειγματοληψίας του πειράματος, πατήστε "Τρέχοντα Πειράματα" από την κυρίως οθόνη. Πατήστε στο εικονίδιο με το Ρολόι, δίπλα από το όνομα του πειράματος που θέλετε να αλλάξετε. Αυτό θα σας μεταφέρει στην ίδια οθόνη που παρουσιάστηκε όταν λάβατε μέρος στο πείραμα, αρχικά.

Τερματισμός συμμετοχής σε ένα Πείραμα

Εάν θέλετε να τερματίσετε την συμμετοχή σας σε ένα πείραμα, πατήστε "Τρέχοντα Πειράματα" από την κυρίως οθόνη. Πατήστε το εικονίδιο του σήματος "Stop" δίπλα από το όνομα του πειράματος που θέλετε να τερματίσετε. Αυτό θα σταματήσει οποιαδήποτε δειγματοληψία. Τα δεδομένα σας θα παραμείνουν διαθέσιμα στον διακομιστή και στη φορητή σας συσκευή, κάτω από το "Προβολή Δεδομένων", αλλά δε θα λαμβάνετε ειδοποιήσεις για να απαντήσετε ξανά, και αυτό το πείραμα δε θα βρίσκεται στη λίστα με τα "Τρέχοντα Πειράματα"

Εξερεύνηση των δεδομένων σας

Από φορητή συσκευή

Από την κυρίως οθόνη, επιλέξτε "Προβολή Δεδομένων". Αυτό θα σας οδηγήσει στον εξερευνητή δεδομένων. Μπορείτε να δείτε τα δεδομένα σας με 4 τρόπους, αναλόγως με τον τύπο τους:

  • Τάσεις: Μπορείτε να δείτε πως μία μεταβλητή του πειράματος μεταβάλλεται με την πάροδο του χρόνου.
  • Συσχετισμοί: Μπορείτε να εξερευνήσετε πως δύο μεταβλητές αλληλεπιδρούν.
  • Διανομή: Μπορείτε να δείτε το φάσμα δεδομένων και τις τάσεις.

Στον διακομιστή/ιστοσελίδα

Μπορείτε να μεταβείτε στην ιστοσελίδα και να επιλέξετε "Τα Πειράματά μου", για να δείτε τα δεδομένα από όλα τα πειράματα στα οποία λάβατε μέρος. Μπορείτε να δείτε απλά διαγράμματα, ακατέργαστα δεδομένα, και να εξάγετε τα δεδομένα για χρήση με άλλα εργαλεία που επιτρέπουν πιο εκλεπτυσμένεες αναλύσεις (R, SPSS, SAS, κτλ...).

Δημιουργία και εκτέλεση πειράματος

Βεβαίως, θα θέλετε να δημιουργήσετε τα δικά σας πειράματα για τον εαυτό σας, ή πιθανώς για κάποιο πληθυσμό εάν είστε κοινωνικός επιστήμονας. Ο σκοπός του Paco είναι να διεξάγετε τη δική σας έρευνα!

Η δημιουργία πειραμάτων είναι εύκολη, αλλά δεν είναι εύκολη σε φορητή συσκευή. Απλά πηγαίνετε στην ιστοσελίδα του διακομιστή από τον υπολογιστή ή τάμπλετ σας (η διεύθυνση βρίσκεται κάτω από το "Επικοινωνήστε μαζί μας" της κυρίως οθόνης) Όταν φτάσετε εκεί, πατήστε "Διαχείρηση πειραμάτων" και "Δημιουργία νέου" για να δημιουργήσετε ένα νέο πείραμα.

Μόλις το δημιουργήσετε, θα είναι διαθέσιμο στην φορητή σας συσκευή. Από την φορητή σας συσκευή, πηγαίνετε στο "Εύρεση πειραμάτων", επιλέξτε "Επαναφόρτωση Πειραμάτων", και θα δείτε το νέο σας πείραμα. Μπορείτε να λάβετε μέρος, ως συνήθως.

Αποστολή ανατροφοδότησης

Επικοινωνήστε σχετικά με το εργαλείο Paco

Εάν θέλετε να επικοινωνήσετε μαζί μας σχετικά με το Paco, από την κυρίως οθόνη επιλέξτε "Επικοινωνήστε μαζί μας". Ακολούθως επιλέξτε "Στείλτε email στην ομάδα Paco", για να μας στείλετε ένα email.

Επικοινωνήστε σχετικά με ένα συγκεκριμένο πείραμα

Ως δημιουργοί του Paco, δεν σχετιζόμαστε με οποιοδήποτε πείραμα, εκτός αυτών που εμείς οι ίδιοι δημιουργήσαμε. Όταν λαμβάνετε μέρος σε ένα πείραμα, το email του δημιουργού εμφανίζεται στην περιγραφή. Μπορείτε επίσης να το βρείτε κοιτάζοντας στο πείραμα, στην ιστοσελίδα. Παρακαλούμε επικοινωνήστε μαζί τους για οποιεσδήποτε ερωτήσεις σχετικά με το συγκεκριμένο πείραμα.

Ορισμοί όρων που χρησιμοποιεί το Paco
ESM
Η Πειραματική Μεθοδολογία Δειγματοληψίας (Experiential Sampling Methodology) είναι μία τεχνική που ιδρύθηκε από τον Mihalyi Csikszentmihalyi για να λαμβάνεται μια καλύτερη εικόνα για κάποιο άτομα, λαμβάνοντας τυχαία δείγματα. Αυτό αποσκοπεί στην εξάλειψη της προκατάληψης ανάληψης.
Πείραμα
Το όνομα για ένα ξεχωριστό καταγραφέα Paco που ίσως θέλετε να δημιουργήσετε ή να λάβετε μέρος σε αυτό. Τα ονομάζουμε πειράματα επειδή πιθανώς να διεξάγετε έξυπνα πειράματα παρεμβάσεων, που ίσως ερωτούν τον εαυτό σας ερωτήσεις σε περιοδική βάση για μία προσέγγιση διαφορετική από την συνήθες, και θα μπορούσε να θεωρηθεί ένα πείραμα αυτογνωσίας. Εντάξει, δεχόμαστε καλύτερα ονόματα! Στείλτε μας τις εισηγήσεις σας!
Δειγματοληψία
Η διαδικασία συλλογής δεδομένων, συνήθως σε προγραμματισμένη ώρα. Το κινητό σας θα σας ειδοποιεί συνήθως και θα σας ζητά να απαντήσετε σε ένα πείραμα. Η απάντησή σας, είναι ένα "δείγμα".
