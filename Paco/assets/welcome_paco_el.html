Καλωσορίσατε στο PACO - τον σύντροφο προσωπικής ανάλυσης (Personal Analytics COmpanion)
"Ὁ δὲ ἀνεξέταστος βίος οὐ βιωτὸς ἀνθρώπῳ" "Γνῶθι σεαυτόν"

Πιστεύουμε ότι αυτά είναι σοφά λόγια.

Το Paco είναι πλατφόρμα που σκοπεύει να σας βοηθήσει να αποκτήσετε "Γνῶθι σεαυτόν". Εσείς επιλέγετε τι θέλετε να παρακολουθήσετε, και ποιός το βλέπει.

Επίσης, το Paco σας επιτρέπει να δημιουργείτε και να λαμβάνετε μέρος σε έρευνες συμπεριφορικής επιστήμης, μαζί με άλλους ανθρώπους για να εξελίξετε την επιστήμη.

Μερικά κόλπα για να ξεκινήσετε:

1) Πατήστε 'Εύρεση Πειραμάτων' για να βρείτε υφιστάμενα πειράματα. Δοκιμάστε μερικά.

2) Πατήστε 'Τρέχοντα Πειράματα' για να διαχειριστείτε τα πειράματα στα οποία λαμβάνετε μέρος.

3) Πηγαίνετε στην ιστοσελίδα του διακομιστή στο http://pacoapp.com/ για να δημιουργήσετε τα δικά σας πειράματα, ή να αντιγράψετε και να επεξεργαστείτε υφιστάμενα

4) Πατήστε 'Εξερεύνηση δεδομένων' για να δείτε τάσεις των δεδομένων σας.

Αυτή είναι μια πρώιμη έκδοση, αλλά ελπίζουμε να σας αρέσει και να μας ενημερώσετε για τις εμπειρίες σας και πως το χρησιμοποιείτε.

Καλή επιτυχία!
